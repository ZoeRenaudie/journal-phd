### **1. Le problème**

"Imaginez : vous cherchez des infos sur les expositions d'art noir au Canada. Vous trouvez un PDF ici, une photo là, un article ailleurs. Mais impossible de voir l'ensemble, de faire des liens, de comprendre l'histoire complète.

Pourquoi ? Parce que toutes ces infos sont **séparées**, comme des livres éparpillés dans différentes bibliothèques."

------

### **2. La solution : le web sémantique**

"Le web sémantique, c'est simple : au lieu de garder l'information dans des documents isolés, on crée des **données reliées**.

Pensez à Wikipedia. Vous cliquez sur un mot souligné, ça vous amène à une autre page, puis une autre. Tout est connecté.

Le web sémantique, c'est pareil, mais pour les **données structurées**. Pas juste du texte libre, mais de l'information organisée qu'un ordinateur peut comprendre et relier."

------

### **3. Comment ça marche : les adresses uniques**

"Chaque concept reçoit une **adresse unique** sur le web — on appelle ça un IRI.

Par exemple, sur Wikidata :

- Le concept **'exposition d'art'** = Q464980
- **'Art Gallery of Ontario'** = Q1075
- **'Toronto'** = Q172

C'est comme une adresse permanente. Peu importe qui parle d'une exposition à Toronto, si on utilise tous ces mêmes codes, on parle de la **même chose**."

------

### **4. Exemple concret : "Here We Are Here"**

"Prenons **'Here We Are Here: Black Canadian Contemporary Art'**, l'exposition historique de 2015 à l'AGO.

**Méthode classique :** Un catalogue PDF quelque part. Si vous ne savez pas qu'elle existe, vous ne la trouvez pas.

**Avec le web sémantique :**

On lui donne une adresse unique, puis on crée des liens :

```
Here We Are Here
  → est une exposition d'art
  → a eu lieu à l'Art Gallery of Ontario
  → à Toronto
  → en 2015-2016
  → organisée par Julie Crooks
  → avec les artistes : Michèle Pearson Clarke, Sandra Brewster...
  → porte sur l'art noir canadien
```

Maintenant quelqu'un peut chercher :

- 'Expositions d'art noir au Canada' → ça apparaît
- 'Michèle Pearson Clarke' → on voit où elle a exposé
- 'Art Gallery of Ontario' → on voit leur historique

Tout est **connecté**."

------

### **5. Créer des réseaux de mémoire**

"Le plus puissant : plusieurs personnes peuvent documenter la même exposition depuis différents endroits.

- L'**AGO** documente les aspects officiels
- **India-Lynn** documente les liens avec les centres autogérés
- Les **artistes** documentent leur expérience
- Les **communautés** ajoutent leurs mémoires

Si tout le monde utilise les mêmes adresses (IRIs), ces infos **se relient automatiquement**.

C'est une **archive distribuée** : pas besoin de tout centraliser, mais tout reste trouvable."

------

### **6. Documenter l'effacement**

"Et voilà le coup de génie pour ma recherche : on peut aussi documenter **ce qui manque**.

Par exemple :

- Pourquoi cette exposition n'existait pas avant 2015 ?
- Quelles œuvres n'ont pas été achetées après ?
- Quels artistes ont été exclus pendant des décennies ?

Je peux créer une **cartographie des effacements** : montrer les patterns, les systèmes d'exclusion, les moments de rupture."

------

### **7. Les limites (important !)**

"Attention : ce n'est **pas magique**.

Le web sémantique reproduit des biais :

- Plus d'hommes que de femmes dans les données
- Plus d'Occident que de Sud global
- Les catégories elles-mêmes sont coloniales

Mon travail, c'est de :

- **Co-construire** les catégories avec les communautés concernées
- Documenter **qui** documente (et ses limites)
- Respecter le droit de certaines pratiques à rester **opaques**

C'est un outil, pas une solution miracle. Mais c'est un outil **puissant** si on l'utilise de manière critique."

------

### **8. Pourquoi ça change tout**

"Pour moi, le web sémantique permet de créer des **archives vivantes**.

'Here We Are Here' n'est plus juste une expo du passé. C'est un **nœud** dans un réseau qui continue de grandir :

- De nouvelles recherches s'y connectent
- De nouvelles mémoires s'y ajoutent
- De nouvelles questions émergent

C'est ça, 'conserver l'éphémère' : pas figer dans le formol, mais créer les conditions pour que ça **persiste et se transforme**.

Comme dit Sara Ahmed : montrer comment les choses persistent, **même quand on ne leur donne pas de place**."
